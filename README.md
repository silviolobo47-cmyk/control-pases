# Control de Pases — DGT
**Dirección General de Transporte — Sección Discapacidad**

App PWA para registrar entregas de pases de discapacidad desde el celular.

## 🚀 Deploy en GitHub Pages

### Paso 1 — Crear el repositorio
1. Ir a [github.com/new](https://github.com/new)
2. Usuario: `silviolobo47-cmyk`
3. Nombre del repo: `control-pases`
4. Visibilidad: **Public**
5. Click en **Create repository**

### Paso 2 — Subir los archivos
Subir estos 5 archivos al repo:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

Para subirlos: en el repo vacío, click en **"uploading an existing file"** → seleccioná los 5 archivos → **Commit changes**

### Paso 3 — Activar GitHub Pages
1. Ir a **Settings** del repo
2. Sección **Pages** (menú izquierdo)
3. Source: **Deploy from a branch**
4. Branch: **main** / **(root)**
5. Click **Save**

### Paso 4 — URL final
```
https://silviolobo47-cmyk.github.io/control-pases/
```
Disponible en ~2 minutos después de activar Pages.

## 📱 Instalar en el celular

**Android (Chrome):**
- Abrir el link → aparece banner "Instalar" → tocar → ícono en pantalla de inicio

**iPhone (Safari):**
- Abrir el link → botón Compartir → "Agregar a pantalla de inicio"

## 🔒 Seguridad de los datos
- Cada registro se guarda automáticamente en el celular al tocarlo "Agregar"
- Funciona sin internet (offline completo)
- Los datos sobreviven cierres, reinicios y falta de batería
- Para pasar a la PC: pestaña Archivo → Descargar .xlsx → compartir por WhatsApp/Drive
